# 飞书小程序部署指南（简化版）

## 不用云函数！用云托管一键部署

### 什么是云托管？

云托管 = 帮你把整个后端项目部署到飞书服务器
- 不用单独配置每个函数
- 不用改代码
- 一键上传整个项目

---

## 部署步骤

### 步骤 1：打包项目

在终端运行：

```bash
cd /Users/boo/.openclaw/workspace/insurance-activity-tool-backend

# 运行打包脚本
bash scripts/deploy.sh
```

这会创建一个 zip 包在 `/tmp/feishu-deploy-*.zip`

---

### 步骤 2：打开飞书开发者工具

1. 确保已打开：`/Applications/飞书开发者工具.app`
2. 扫码登录
3. 选择你的应用：`cli_a95a6b370af8dcc8`

---

### 步骤 3：找到云托管

在飞书开发者工具顶部菜单，找：
- **"云托管"** 或 **"云开发"** 标签
- 或者在左侧菜单找 **"后端服务"**

如果找不到，用这个替代方案：

## 替代方案：直接使用当前后端

### 方案 A：用本地后端测试（推荐用于开发）

1. 确保后端在运行：
   ```bash
   npm run dev
   ```

2. 小程序 `app.js` 中的 `apiBase` 已经是 `http://localhost:3000`

3. 在飞书开发者工具中：
   - 点击 **"小程序"**
   - 导入小程序目录：`/Users/boo/.openclaw/workspace/insurance-activity-tool-lark-miniprogram`
   - 点击 **"预览"** 或 **"调试"**
   - 测试功能

### 方案 B：部署到飞书云托管（生产环境）

等小程序调试完成后，再部署后端到云托管。

## 飞书开发者工具界面说明

打开后你应该看到这些标签：

| 标签 | 用途 |
|------|------|
| 小程序 | 开发和预览小程序（你现在用的）|
| 云托管 | 部署后端服务（暂时不用管）|
| 调试 | 查看日志和调试信息 |
| 设置 | 配置应用信息 |

你现在只需要用 **"小程序"** 标签就够了！
