# 飞书云托管开通指南

## 方法 1：在飞书开发者工具中开通

1. 打开飞书开发者工具
2. 点击你的应用（或创建新应用）
3. 找 **"云托管"** 或 **"云开发"** 标签
4. 点击 **"开通"** 或 **"启用"**
5. 按提示完成开通

## 方法 2：在飞书开放平台开通

1. 打开浏览器访问：https://open.feishu.cn
2. 扫码登录
3. 进入你的应用：`cli_a95a6b370af8dcc8`
4. 左侧菜单找 **"云托管"** 或 **"云服务"**
5. 点击开通

## 方法 3：使用云函数（替代方案）

如果云托管不可用，使用云函数：

1. 在飞书开发者工具中
2. 点击 **"云函数"** 标签
3. 创建新云函数
4. 上传后端代码

## 如果以上都不可用

**替代方案：使用外部服务器**

1. 把后端部署到任何云服务器（阿里云、腾讯云等）
2. 修改小程序 `app.js` 中的 `apiBase` 为服务器地址
3. 小程序继续用飞书，后端用外部服务器
